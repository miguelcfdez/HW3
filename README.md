# Homework 3: Unique Words Comparison
In this exercise: 
- I defined a function that downloads a book from an URL (Project Gutenberg)
- I wrote a code that counts the unique words in each book.
- Then, the code will compare them and return which of them has more unique words

## Concepts Used
- Files (open, reading line by line)
- Strings (replace, lower, split)
- Dictionaries
- Loops (for)
- Functions
- Conditional statements (if/elif/else)
- Try/Except
- Requests library (to download the books)
